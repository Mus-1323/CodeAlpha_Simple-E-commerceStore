
A full-stack E-Commerce web application built using Node.js, Express.js, MongoDB, EJS, and Passport.js.  
The project includes user authentication, product listings, shopping cart, and order management features.

---

# Features

✅ User Signup & Login Authentication  
✅ Product Listing Page  
✅ Product Details Page  
✅ Shopping Cart  
✅ Order Placement  
✅ My Orders Page  
✅ MongoDB Database Integration  
✅ Protected Routes using Passport.js  
✅ Session-Based Authentication

# Installation & Setup

## Clone Repository

```bash
git clone https://github.com/Mus-1323/CodeAlpha_Simple-E-commerceStore.git
```

## Move to Project Folder

```bash
cd CodeAlpha_Simple-E-commerceStore
```

## Install Dependencies

```bash
npm install
```

## Run Server

```bash
node app.js
```

---

# MongoDB Setup

Make sure MongoDB is running locally.

```bash
mongodb://127.0.0.1:27017/E-commerce
```

# Main Routes

| Route | Description |
|------|-------------|
| `/product` | Show all products |
| `/product/:id` | Product details page |
| `/signup` | User registration |
| `/login` | User login |
| `/logout` | Logout user |
| `/cart` | Shopping cart |
| `/order` | Place order |
| `/my-order` | User orders |
